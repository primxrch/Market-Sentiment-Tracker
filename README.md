# Real-Time Financial News Sentiment Engine

📌 **Interactive Project Overview:** An asynchronous, production-minded data application built to stream breaking global news items over the web, execute real-time algorithmic NLP classification, and output analytical business insights.

## ⚙️ App Pipeline Framework
1. **Asynchronous Ingestion:** Takes user text inputs and maps outbound requests to global news index servers via standard REST APIs.
2. **Natural Language Processing:** Processes raw string payloads using text mining algorithms to scale textual content into objective mathematical polarity matrices ranging from -1.0 to +1.0.
3. **Presentation Interface:** Packages backend evaluation logs into a data delivery dashboard built entirely in Python using **Streamlit** and **Plotly**.

## 🛠️ Software Toolstack
- **Language Stack:** Python 3.12+ Natively Configured
- **UI Architecture:** Streamlit Web Engine
- **Data Engineering:** Pandas Matrix Framework
- **Data Visualization:** Plotly Express Responders
- **Text Processing Algorithmic Engine:** TextBlob Natural Language Processing
